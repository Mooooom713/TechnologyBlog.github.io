开博首盘点--非常顺手的编程字体

一些废话

为什么会写这篇博客呢？主要是日常做前端开发会一直和编辑器打交道，看着那些自己辛辛苦苦码上去的代码，写的优雅除了自己本身的写法以外，我觉得字体也很重要。好看的字体会让人编码的时候都赏心悦目呀！话不多说，主要还是盘点几个我最喜欢的编程字体。

operator mono

  • 一句话总结这个字体和我的关联–只是因为在字群中多看了你一眼

  • 上图：

  • 是不是看上去就十分的让你怦然心动呢？毕竟这也是付费字体，但是平穷的码农还是要动动小脑筋

  • 神奇传送门

  • tips：这些传送门随时可能会失效，心动不如行动

Fira Code

我想对颜值有要求的码农对这个字体应该是如雷贯耳了，毕竟颜值高又开源，岂不是美滋滋？

  • 效果图：

  • 真的也非常好看呀！！！(♥∀♥)

  • 神奇传送门

  • tips：这个字体我在Windows 10 的vs code上按照官网设置了之后还是出不来连字，有点崩溃的时候看见了官网上的一个issue传送门按照上面大神的介绍用Cleartype处理之后就完全OK了，哭泣！！！

Lucida Saans Unicode

这个字体是我windows电脑上自带的字体，在尝试了众多字体之后真心觉得这个字体非常适合拿来敲代码

  • 效果图：

Ink Journal

这个字体有点花体的味道，但是我也不知道算不算，毕竟不是专业的。不过这个字体也是真的很好看啦！！！也是我windows电脑自带的字体

  • 效果图：

endinng

以上几种字体都是我在字体海洋里，各种尝试各种浪之后选出来我觉得比较好用的，当然曾经我也用花体、手写体各种各样，仿佛回到了葬爱的年代，但是本来编程用的字体也就是自己看上去爽就好了，神清气爽才能码出精彩、码出未来嘛~
